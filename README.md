# Terra Client

A lightweight, modern Minecraft launcher built with Electron.

## What It Is

Terra Client is a launcher for Minecraft: Java Edition that lets you:

- Launch the game with custom RAM and version settings
- Sign in with your Microsoft accounts
- Manage multiple Minecraft versions from one place
- Use a clean, simple interface without bloat

## Why We Built It

The official launcher works, but it lacks flexibility. Most third-party launchers are either outdated, bloated, or built for modpacks first. We wanted something that:

- **Just works** – no complicated setup
- **Stays out of your way** – launch and play

## What We Need

To make the launcher work properly, we need:

1. **Minecraft API Access** – to authenticate users and retrieve their profile information
